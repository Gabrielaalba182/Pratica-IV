# Projeto Adaptado - Entrega IV

Este repositório contém a versão adaptada do site original 'Pratica-III---desenvolvimento-java'.

Principais adaptações:
- Acessibilidade (WCAG 2.1 AA): skip-link, roles, focus styles, keyboard navigation, aria attributes.
- Modo escuro e alto contraste com armazenamento em localStorage.
- Arquivos minificados e prontos para deploy em /dist.
- Workflow GitHub Actions para deploy no GitHub Pages (.github/workflows/deploy.yml).

Deploy: coloque conteudo de /dist no GitHub Pages (ou use o workflow incluído).
